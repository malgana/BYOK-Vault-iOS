# Руководство по добавлению иконок платформ

## Что было реализовано

### ✅ Система иконок с тремя уровнями:

1. **Предустановленные платформы** — иконки из Assets (PNG 250×250px)
2. **Кастомные платформы с иконкой** — пользователь загружает свою иконку
3. **Fallback** — автоматическая генерация иконки с первой буквой в цветном круге

### ✅ Функционал:

- Автоматическое сжатие загруженных иконок до 250×250px
- PhotosPicker для выбора изображения при создании новой платформы
- Хранение кастомных иконок в Swift Data
- Автоматическая адаптация для темной темы (легкая обводка)
- Опциональная загрузка иконки — не обязательна для пользователя

## 📋 Что нужно сделать вручную

### Добавить PNG иконки в Assets (250×250px)

Вам нужно добавить PNG файлы размером **250×250px** для следующих платформ:

1. **Anthropic** → `KeyVault/Assets.xcassets/anthropic.imageset/anthropic.png`
2. **OpenAI** → `KeyVault/Assets.xcassets/openai.imageset/openai.png`
3. **Google AI** → `KeyVault/Assets.xcassets/google-ai.imageset/google-ai.png`
4. **Hailuo** → `KeyVault/Assets.xcassets/hailuo.imageset/hailuo.png`
5. **DeepSeek** → `KeyVault/Assets.xcassets/deepseek.imageset/deepseek.png`
6. **Reve AI** → `KeyVault/Assets.xcassets/reve-ai.imageset/reve-ai.png`
7. **GitHub** → `KeyVault/Assets.xcassets/github.imageset/github.png`
8. **Google Image Search** → `KeyVault/Assets.xcassets/google-image-search.imageset/google-image-search.png`

### Как добавить иконки:

#### Способ 1: Через Finder
1. Подготовьте PNG файлы размером 250×250px
2. Переименуйте файлы согласно списку выше
3. Скопируйте их в соответствующие папки `.imageset`

#### Способ 2: Через Xcode (рекомендуется)
1. Откройте проект в Xcode
2. Перейдите в `Assets.xcassets`
3. Найдите нужный Image Set (например, `anthropic`)
4. Перетащите PNG файл в ячейку "Universal"

---

## 🎨 Требования к иконкам

- **Размер**: 250×250px (будут автоматически масштабироваться)
- **Формат**: PNG с прозрачным фоном (предпочтительно)
- **Цвет**: Цветные иконки (не нужно делать отдельные версии для темной темы)
- **Качество**: Четкие, без артефактов

---

## 🔄 Автоматическая адаптация для темной темы

Система автоматически добавляет тонкую обводку иконкам в темной теме для лучшей читаемости. Вам не нужно создавать отдельные версии для светлой и темной темы.

---

## 👤 Как пользователь добавляет иконку

1. При создании нового ключа выбирает "New" в списке платформ
2. Вводит название новой платформы
3. Появляется секция "Иконка (опционально)" с кнопкой "Добавить иконку"
4. Нажимает на кнопку и выбирает изображение из галереи
5. Изображение автоматически сжимается до 250×250px
6. Показывается preview выбранной иконки
7. При сохранении ключа иконка сохраняется вместе с платформой

---

## 🔍 Fallback система

Если у платформы нет иконки (ни в Assets, ни загруженной пользователем):
- Автоматически генерируется иконка с первой буквой названия
- Цвет выбирается детерминированно на основе названия
- Первая буква отображается белым цветом в цветном круге
